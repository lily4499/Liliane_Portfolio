Set up the deployement of an application on Amazon Elastic Kubernetes Service (EKS)

  • POM file inside the code in github
  • Jenkins build artifact using maven and store it in his workspace
  • Dockerfile inside the code in github,that copy artifact from jenkins to Tomcat environment
  • Jenkins to login to AWS: we install aws cli , create a user on AWS, then configure his credential on jenkins
  • Jenkins uses some plugins to build a docker image, login to AWS and push it to ECR
  • Jenkins to deploy to EKS: we install kubectl, eksctl, make jenkins the owner and group of the kube config file
